Contact-Information
"Contact Information" is created to demonstrate get/add/edit/delete of contacts.

Steps to run this project:

1. restore packages, run below command to do it. 
	npm install
2. Build solution in VS2015/17
3. Hit cntl+F5 to launch the web portal
4. Click on 'Contact-Information' to navigate.
5. Validations in place: 
	a. Required field 
	b. Minimum length for input controls 
	c. Email validation 
	d. Phone number validation
